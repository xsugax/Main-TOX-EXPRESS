# ⚡ Tidio Live Chat - Quick Reference

## 🚀 5-Minute Setup

### Get Your Tidio ID in 2 Steps:
1. Sign up: https://www.tidio.com/ (Free)
2. Copy ID from: Dashboard → Settings → Channels → Website → Project ID

### Add to Your Website in 1 Step:
**In `index.html` line ~676, replace:**
```
YOUR_TIDIO_ID
```
**With your actual ID** (e.g., `a1b2c3d4e5f6`)

🎉 **Done!** Reload your website - chat widget appears instantly!

## 🔧 Common Actions

### Enable Agent
1. Dashboard → Team → Add Agent
2. Invite by email
3. They accept invite
4. They set status to "Available"

### Set Business Hours
1. Dashboard → Automation → Business Hours
2. Set timezone & hours
3. Create "outside hours" message
4. Save

### Add Welcome Message
1. Dashboard → Appearance
2. Edit "Greeting" message
3. Make it friendly & helpful
4. Save

### View All Chats
1. Dashboard → Conversations
2. See all past chats
3. Search by customer name/date
4. Click to view details

## 🚨 Troubleshooting Quick Fixes

| Problem | Fix |
|---------|-----|
| Chat not showing | Hard refresh: Ctrl+Shift+R |
| No messages received | Check agent status = "Available" |
| Widget in wrong spot | Adjust in Tidio Settings → Appearance |
| Slow responses | Add more agents or enable AI Bot |
| Missing chat history | Check filters in Dashboard |
| SSL warning | Ensure https:// on your domain |
